# 苍穹研发问题原因分析技能 (cosmic-issue-analysis)

这是一个专为 **金蝶云苍穹 (Kingdee Cloud Cosmic)** 研发环境设计的智能助手技能。它旨在帮助开发与运维人员快速诊断系统故障、分析日志堆栈、定位部署异常，并提供结构化的根因分析报告及解决方案。

## 🚀 核心功能

- **自动化日志解析**：一键分析应用日志、数据库堆栈及补丁安装错误。
- **结构化分析报告**：按照“问题定义-数据证据-根因分析-解决方案-预防措施”的专业标准输出报告。
- **内置典型案例库**：集成 20+ 真实研发与运维场景，支持快速匹配已知问题。
- **部署指引与新手教程**：包含补丁包安装规范及常见坑点说明。

## 📂 典型案例库亮点

技能内置了丰富的实战案例，特别针对以下高频问题进行了深度优化：

- **案例六：MC 补丁包部署主键冲突**
  - 支持分析表单级（`t_meta_formdesign`）及应用级（`t_meta_bizapp`）冲突。
  - **核心场景**：识别开发环境删除重建应用导致的主键 ID 变化冲突（编码相同但 ID 不同）。
- **案例二十：分库标识（RouteKey）缺失**
  - 诊断执行 SQL 脚本时提示“数据中心未配置数据源”的问题，提供标准的脚本命名规范。
- **案例二十一：类加载异常（not find）**
  - 针对 JAR 包更新后未重启微服务导致的部署失败提供止血方案（如 `BalTbModifyTimeUpgrade` 异常）。

## 🛠️ 安装与使用

### 安装
1. 将本仓库克隆或下载到本地。
2. 将文件夹放置在 Qoder 技能目录下：`{home}/.qoderwork/skills/`。

### 使用方法
在 Qoder 对话框中直接输入指令：
```bash
/cosmic-issue-analysis <日志文件路径、错误代码或报错截图>
```
